# WingWing PPL Textbook: Master Structure & Mapping

This document defines the organizational structure for the WingWing Private Pilot textbook. It maps the Jeppesen chapter structure (which WingWing follows) to the FAA PHAK source material.

## Jeppesen Layout Template

Each section follows this structure:

### Opening
1. **Section Header** - "SECTION A: [Title]"
2. **Introductory Paragraph** - Historical hook or context (2-3 sentences)
3. **Overview** - What this section covers

### Main Content
4. **CAPS HEADINGS** - Major topic divisions
5. **Body Text** - Short paragraphs (2-5 sentences)
6. **Figure References** - "[Figure X-X]" inline
7. **Sidebars** - Historical context, technical deep-dives
8. **Bold Terms** - Key vocabulary on first use
9. **Memory Aids** - Acronyms like ARROW, IMSAFE

### Closing
10. **Summary Checklist** - Bulleted key takeaways
11. **KEY TERMS** - Two-column vocabulary list
12. **QUESTIONS** - Mixed formats:
    - Diagram identification
    - Short answer
    - True/False
    - Matching
    - Multiple choice
